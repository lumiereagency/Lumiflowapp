## 🚀 Como rodar localmente

O código foi exportado pelo **Figma Make**, que não inclui automaticamente os arquivos de configuração (`package.json`, `vite.config.ts`, `tsconfig.json`, etc.).

Para rodar o projeto localmente, use o **figma-make-local-runner**:

### Passo a passo

**1. Clone este repositório**
```bash
git clone https://github.com/lumiereagency/LUMIFLOW.git
```

**2. Clone o figma-make-local-runner**
```bash
git clone https://github.com/likang/figma-make-local-runner.git
cd figma-make-local-runner
```

**3. Copie os arquivos deste repositório para a pasta `src`**
```bash
cp -r ../LUMIFLOW/* src/
```

**4. Instale as dependências e rode**
```bash
npm install
npm run dev
```

**5. Acesse no navegador**
```
http://localhost:5173
```
